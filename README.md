# `Social Media Solution(Front-end)`

Complete front end of social media solution on react.js

### `.env file`

```
REACT_APP_DEFAULT_API_URL=
REACT_APP_GOOGLE_CLIENT_ID=
```

### `running commands(macOS)`

```
npm install
npm start
```
